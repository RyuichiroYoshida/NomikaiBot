# NomikaiBot

## 概要

飲み会募集DiscordBot

## MVP

- Discordスラッシュコマンドの作成
- 飲み会募集のメッセージ送信
- 飲み会参加者の情報を主催者が確認できる
- 参加者が日時、場所、参加者を把握できる

## 技術選定について

### DiscordBot

現在想定している利用者・参加者は同じDiscordサーバー内にいるメンバーのみ、

Discord以外の連絡手段を持たないユーザーが多いため、募集の場や連絡確認などを含めDiscordで完結させるのが適切と判断

### Golang

Goのコード設計の練習の意も含めている

DiscordBotのGO SDKを利用

### MySQL

コードログ保存、過去の開催情報を保存
