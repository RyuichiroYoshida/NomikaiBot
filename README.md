# NomikaiBot

## 概要

飲み会募集Webサービス

- 調整さん改みたいなツール
- 調整さんに場所の候補と時間の希望を投げれるようにする

## MVP

- 飲み会募集のメッセージ送信
- 飲み会参加者の情報を主催者が確認できる
- 参加者が日時、場所、参加者を把握できる

## ドメイン

- 参加者一覧(人数)が必要
- 主催者が日時・場所を登録する
- 募集をDiscordへ送信する
- 募集は削除できる
- 募集中かの状態を確認できる

## 処理の流れ

1. ブラウザ上から、主催者が日時・場所を指定して募集を作成する
2. Discordで告知される
3. 参加者は告知文から、参加希望を名前を併せて出す

## 技術選定について

### フロントエンド

- React / TypeScript

### バックエンド

API本体、ドキュメント

- Go / TypeSpec / OpenAPI

### DB

- MySQL コードログ保存、過去の開催情報を保存

### デプロイ

- AWS
